Продажи проектов

Обработка поступившего запроса на разработку проекта, подготовка коммерческого предложения (КП) по проекту, заключение договора на разработку и реализацию проекта, получение полной или частичной оплаты выполненных работ.

Перевод проекта в стадию разработки, передача его в проектный отдел компании, для выполнения работ по разработке проекта. Финальной стадией процесса является получение полной оплаты за разработку и работы по проекту и, при необходимости, получение обратной связи по результатам выполненных работ.

Для работы с заказами компании разработана специализированная карточка сделки с необходимыми для обработки заказа полями. Предусмотрена обязательность заполнения ключевых полей сделки при переходе на стадии обработки. Для работы с карточкой заказа, кроме базовый полей, созданы специальные пользовательский поля для заполнения данных по проекту и характеристикам заказа клиента.

В карточке сделки реализовано подключение дополнительного функционала расчета предварительной оценки работ по проекту и данных разработки проекта

  • Оценка работ по проекту

    Оценка работ по проекту

    Данный инструмент предназначен для составления предварительного списка этапов разработки, работ по проекту, сроков и стоимости работ при составлении коммерческого предложения (КП) для заказчика, в случае если эти виды работ и этапов не были занесены как услуги в товарном каталоге CRM.

  • План-график разработки проекта

    План-график разработки проекта

    Составление и ведение План-графика работ по проекту с автоматическим отслеживание сроков выполнения запланированных этапов работ. Возможность создания связанных задач по утвержденному план-графику работ.

  • Документы по проекту

    Документы по проекту

    Подключения   карточку разработки ведения реестра документов по проекту. Позволяет получить доступ из карточки проекта ко всем необходимым документам и договорам по проекту с описанием списка работ, подрядчиков по работам и т.д.

  • Расходы и платежи

    Расходы и платежи

    Фиксация в карточке проекта расходов и платежей по проекту (финансы по проекту), как запланированных, так и уже произведенных с автоматическим расчетом в поле карточки суммы текущих расходов по проекту, запланированных расходов по проекту и текущей доходности проекта.

  • Подрядчики по проекту

    Подрядчики по проекту

HR отдел (отдел кадров)

Ведение заказа клиента от момента продажи до момента сдачи объекта клиенту. Контроль и мониторинг обработки заказа в смежных отделах: проектирование, производство. Мониторинг выполнения строительных работ на объекте заказчика. При переходе сделки в стадию «Передачи в отдел Реализации» в соответствующем направлении обработки создается карточка проекта с автоматическим получением необходимых данных из сделки.
Для работы менеджера отдела реализации по проекту заказчика реализовано подключение дополнительного функционала в карточке проекта.
